SubspaceBot¶

class SubspaceBot.SubspaceBot(owners_list=None, description=None, debug=False)¶

The bot. Must be connected with connectToServer() and then waitForEvent() must be called frequently for adequate performance.

The typical bot’s mainloop looks like:

while bot.isConnected():
        event = bot.waitForEvent():
        if event.type = ...:
                ...
        elif event.type = ...:
                ...
connectToServer(hostname, port, username, password, arena='#master')¶

Connect to a server using the Subspace protocol.

hostname and port specify the hostname/IP address and port of the server to connect to. username is the user to connect as. password is the combined password and SMod+ password seperated by an asterisk. For example, ‘bot_password*smod_password’. Arena is the name of the arena to join upon entering the zone.

findPlayerByName(name)¶

Find a player by name.

If a player with the exact name is not found, None is returned.

findPlayerByPid(pid)¶

Find a player by PID.

If a player is not found, None is returned.

isConnected()¶
Returns True if the bot is connected to the server, otherwise False.
registerCommand(name, description, allowed_players=None)¶

Register a command with the core.

name is the name of the command, including the ‘!’. description is a short one-line explanation of what the command does, displayed in !help.

allowed_players is a list of players who are allowed to use the command. If the list is None, then all players are allowed to use it.

Returns a unique identifier for the command, to be used in EVENT_COMMAND to identify the command being used.

requestFreqChange(freq)¶

Sends a freq change request to the server.

Note that until you receive an EVENT_CHANGE update for the bot, your freq has not changed.

requestShipChange(ship_type)¶

Sends a ship change request to the server.

Note that until you receive an EVENT_CHANGE update for the bot, your ship has not changed.

sendArenaMessage(message, sound=0)¶

Send a message to the arena with an optional sound.

message can be a list of messages to send.

sendFreqMessage(freq, message)¶

Send a freq message.

message can be a list of messages to send.

sendPrivateMessage(player, message, sound=0)¶

Send a private message to player.

player can be a Player object or the player’s name.

message can be a list of messages to send.

sendPublicMessage(message, sound=0)¶

Send a public message with an optional sound.

message can be a list of messages to send.

Public messages are sent to the server reliably but may not be relayed from the server to all players reliably.

setPosition(x_pos=8192, y_pos=8192, x_vel=0, y_vel=0, rotation=0, status_flags=0, bounty=0, energy=0)¶

Set the bot’s coordinates and other position data and send the new position to the server.

status_flags is made of combinations of STATUS_Xxx.

When this is called a packet is queued for send immediately.

setTimer(seconds, user_data=None)¶

Sets a timer that will generate an EVENT_TIMER event in seconds seconds.

user_data is passed back as event.user_data during EVENT_TIMER.

Returns a unique timer id that is passed back in EVENT_TIMER’s event.id when the timer expires.

waitForEvent()¶

Wait for an event.

A GameEvent class instance is returned, and its type can be found in GameEvent.type. The type will be one of EVENT_Xxx. If the bot is disconnected None will be returned.
